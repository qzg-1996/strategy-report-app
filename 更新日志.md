# 更新日志

## 2024-03-27 重大更新

### 界面全面重构
- 采用现代化单页应用设计
- 左侧导航栏 + 右侧内容区布局
- 顶部深蓝色渐变标题栏
- 圆润的卡片设计，带阴影和悬停效果

### 按钮样式优化
- 圆角按钮设计（8-12px圆角）
- 渐变色按钮（蓝、金、绿）
- 悬停时上浮效果
- 图标+文字组合

### 上传功能改进
- 卡片式上传区域
- 上传成功后显示"已上传"标记
- 统计数字实时更新
- 更好的错误提示

### 数据可视化
- 统计卡片带图标和颜色区分
- 表格头部渐变背景
- 盈亏数字颜色标识（红/绿）
- 空状态友好提示

### 交互体验
- Toast通知系统（右上角弹出）
- 标签页切换动画
- 加载状态提示
- 表单验证反馈

### 功能模块
1. **数据导入** - 主力合约配置 + 4个数据文件上传
2. **策略管理** - 添加/编辑/删除策略
3. **策略汇总** - 实时统计所有策略盈亏
4. **周报生成** - 填写内容并生成PDF

## 使用说明
1. 双击 `start.bat` 启动系统
2. 浏览器访问 http://127.0.0.1:5000
3. 按顺序上传数据文件
4. 配置策略
5. 生成周报
